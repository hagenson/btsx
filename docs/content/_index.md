# BTSX Help Documentation

Welcome to the BTSX (Better Than SyncX) documentation. BTSX is an IMAP email migration tool that helps you move emails from one IMAP server to another.

## Features

- **Console Application**: Command-line tool for batch email migrations
- **Web Interface**: User-friendly web UI for interactive migrations
- **Real-time Progress**: Live status updates during migration
- **OAuth Support**: Google OAuth integration for secure authentication
- **Folder Preservation**: Maintains folder hierarchy and message flags
- **Secure**: Encrypted credential storage
